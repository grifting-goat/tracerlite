# Tracerlite

I watched a bunch of videos on youtube about raytracing which gave me an unearned since of compentance.

## Scope

My current goals for this project are
- set up the Vulkan pipeline
- simple camera movement
- generate small voxel world
- render each pixel by pathtracing using Amanatides-Woo on a voxel grid (just color no lighting yet)
